2.数据整理¶

  • 从上面可以看到，data总共有1309行数据，而Cabin（客舱位置）这一列只有295个非缺失值，不适合用缺失值插补的方法，因此应删除该列。其他缺失值（Age缺失263个、Fare缺失1个、Embarked缺失2个）根据特点选择用中位数、均值或众数进行插补。

填补缺失值¶

In [40]:
data = data.drop('Cabin', axis=1)                                       #缺失值太多，剔除Cabin变量                
data['Age'][data['Age'].isnull()] = data['Age'].median()        #年龄缺失值用中位数插补
data['Fare'][data['Fare'].isnull()] = data['Fare'].mean()         #票价缺失值用均值插补
data['Embarked'][data['Embarked'].isnull()] = data['Embarked'].mode().values    #上船港口编号缺失值用众数插补
In [44]:
data['Sex'] = pd.factorize(data['Sex'])[0]                           #将性别、上船港口转换为因子
data['Embarked'] = pd.factorize(data['Embarked'])[0]
data = data.drop('Name', axis=1)                                     #简便起见，删除姓名和船票编号
data = data.drop('Ticket', axis=1)
  • 删除姓名Name和船票编号Embarked这并不意味着这两个变量不重要，在很多Titanic数据分析中它们的使用频率还是很高的。对这两个变量的规范处理参考Kaggle数据挖掘竞赛
